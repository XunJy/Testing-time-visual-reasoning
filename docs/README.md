# Documentation

The repository keeps current instructions close to the code and immutable
results close to their experiment.

- [`../README.md`](../README.md): current project overview, results, setup, and
  repository structure.
- [`../experiments/README.md`](../experiments/README.md): authoritative
  method + model + dataset registry.
- `../experiments/<experiment>/README.md`: locked protocol, commands, and
  interpretation for one concrete experiment.
- `../experiments/<experiment>/runs/` or `analysis/`: immutable machine-readable
  artifacts when their size and license allow Git publication.
